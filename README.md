# SIT725 - Applied Software Engineering

## Task 2.2P - Express Web Server

This project uses Node.js and Express to serve a webpage from the public folder.

It also includes a GET endpoint that adds two numbers using query parameters.

## Run the project

Install the dependencies:

`npm install`

Start the server:

`npm start`

Open the webpage:

`http://localhost:3000`

Test the addition endpoint:

`http://localhost:3000/add?a=10&b=5`
